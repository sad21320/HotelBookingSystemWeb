## Описание проекта
Лабораторная работа по разработке ASP.NET Core MVC приложения.
Реализована система для автоматизации работы отеля.

## Функционал
*   **Гости**: Учет постояльцев (ФИО, паспорт).
*   **Номера**: Управление номерным фондом (типы, статусы).
*   **Бронирования**: Оформление заездов, расчет стоимости.
*   **Сотрудники**: Отдел кадров.
*   **Поиск**: Поиск бронирований по фамилии или номеру.

## Технологии
*   ASP.NET Core 8.0 MVC
*   Entity Framework Core (SQL Server)
*   GitHub Actions (CI/CD)
