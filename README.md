# Система бронирования отеля (Вариант 30)

![Build Hotel System](https://github.com/sad21320/HotelBookingSystemWeb/actions/workflows/dotnet.yml/badge.svg)

## Описание проекта
Лабораторная работа по разработке ASP.NET Core MVC приложения.
Реализована система для автоматизации работы отеля с базой данных MS SQL Server.

## Функционал
*   **Гости**: Учет постояльцев (ФИО, паспортные данные).
*   **Номера**: Управление номерным фондом (типы, статусы, цены).
*   **Бронирования**: Оформление заездов, автоматический расчет стоимости.
*   **Сотрудники**: Отдел кадров.
*   **Поиск**: Фильтрация бронирований по фамилии гостя или номеру комнаты.
*   **Кэширование**: Оптимизация запросов (время хранения 300 сек).

## Технологии
*   ASP.NET Core 8.0 MVC
*   Entity Framework Core (Code First)
*   GitHub Actions (CI/CD: сборка под Windows и Linux)
